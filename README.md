# knirpslarsi.github.io

Meine persönliche Website, gebaut mit Jekyll und GitHub Pages.

## Design Features

- **Modern Minimal Design**: Matt schwarz mit Lila Akzenten
- **Animated Title**: Knirpslarsi mit Gradient-Animation auf der Startseite
- **Social Links**: GitHub, Twitch, Discord Links auf der Startseite (ohne Unterstrich)
- **Project Detail Pages**: Klickbare Projekte mit ausführlichen Informationen
- **Auto-formatted Projects**: Projekte aus `_config.yml` werden automatisch formatiert
- **Smooth Animations**: Hover-Effekte und Übergänge
- **Responsive**: Funktioniert auf allen Geräten

## Farben

- Background Dark: `#111111`
- Background Secondary: `#1a1a1a`
- Accent Purple: `#a855f7`
- Accent Purple Light: `#c084fc`

## Social Links konfigurieren

In `_config.yml` unter `social:` deine Links eintragen:

```yaml
social:
  github: "https://github.com/dein-username"
  twitch: "https://twitch.tv/dein-channel"
  discord: "https://discord.gg/dein-server"
