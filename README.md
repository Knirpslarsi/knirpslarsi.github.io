# knirpslarsi.github.io

Meine persönliche Website, gebaut mit Jekyll und GitHub Pages.

## Design Features

- **Modern Minimal Design**: Matt schwarz mit Lila Akzenten
- **Smooth Animations**: Hover-Effekte und Übergänge
- **Responsive**: Funktioniert auf allen Geräten
- **Clean Code**: Einfach zu erweitern und anzupassen

## Farben

- Background Dark: `#0a0a0a`
- Background Secondary: `#1a1a1a`
- Accent Purple: `#a855f7`
- Accent Purple Light: `#c084fc`
