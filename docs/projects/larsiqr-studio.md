<section>
    <div class="container project-content">

        <div class="project-section">
            <h2>Overview</h2>
            <p>
                LarsiQR Studio is a modern desktop QR code generator for Windows.
                It allows you to quickly generate QR codes from URLs or text, export them
                in multiple formats, and even create hundreds of QR codes automatically
                using a CSV batch generation system.
            </p>
            <p>
                Designed with a clean dark user interface and built using Java Swing and FlatLaf.
            </p>
        </div>

        <div class="project-section">
            <h2>Key Features</h2>
            <ul>
                <li><strong>QR Code Generation:</strong> Generate QR codes from URLs or text</li>
                <li><strong>Dark Interface:</strong> Clean and modern dark UI</li>
                <li><strong>Live Preview:</strong> Automatic preview with autoscaling</li>
                <li><strong>Export Formats:</strong> PNG, JPG and SVG support</li>
                <li><strong>Logo Support:</strong> Add custom logos to single QR codes</li>
                <li><strong>Batch Generation:</strong> Generate hundreds of QR codes using CSV files</li>
                <li><strong>Drag & Drop:</strong> Drop URLs, text or CSV files directly into the application</li>
                <li><strong>Update Checker:</strong> Automatic update notifications</li>
                <li><strong>Windows Installer:</strong> Easy installation and setup</li>
            </ul>
        </div>

        <div class="project-section">
            <h2>Technologies Used</h2>
            <div class="tech-tags">
                <span class="tech-tag">Java</span>
                <span class="tech-tag">Java Swing</span>
                <span class="tech-tag">FlatLaf</span>
                <span class="tech-tag">ZXing</span>
                <span class="tech-tag">CSV Processing</span>
            </div>
        </div>

        <div class="project-section">
            <h2>Single QR Code Generation</h2>
            <p>Creating a QR code only takes a few steps:</p>
            <ul>
                <li>Enter a URL or text</li>
                <li>Click <strong>Generate QR Code</strong></li>
                <li>Optionally add a logo</li>
                <li>Select an export format</li>
                <li>Save the QR code</li>
            </ul>
        </div>

        <div class="project-section">
            <h2>Batch QR Code Generator</h2>
            <p>
                LarsiQR Studio can automatically generate large numbers of QR codes
                using CSV files.
            </p>

            <p><strong>Example use cases:</strong></p>

            <ul>
                <li>Product labels</li>
                <li>Event tickets</li>
                <li>Marketing campaigns</li>
                <li>Bulk QR code generation</li>
            </ul>

            <p>
                A ready-to-use CSV template can be downloaded directly inside the application.
            </p>
        </div>

        <div class="project-section">
            <h2>Drag & Drop Support</h2>
            <p>Supported drag & drop actions:</p>

            <ul>
                <li><strong>URL:</strong> Fills the input field automatically</li>
                <li><strong>Text:</strong> Fills the input field automatically</li>
                <li><strong>CSV File:</strong> Starts the batch generator</li>
            </ul>
        </div>

        <div class="project-section">
            <h2>Installation</h2>
            <p>
                Download the latest installer from the Releases section and follow
                the setup wizard.
            </p>

            <ul>
                <li>Desktop shortcut support</li>
                <li>Start Menu integration</li>
                <li>No Java installation required</li>
            </ul>
        </div>

        <div class="project-section">
            <h2>Updates</h2>
            <p>
                The application automatically checks for updates when it starts.
                If a new version is available, a notification will appear and the
                update can be downloaded directly.
            </p>
        </div>

        <div class="project-section">
            <h2>System Requirements</h2>

                <li>No separate Java installation required</li>
            </ul>
